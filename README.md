# Combo.Storage

[![CI](https://github.com/combo-lab/combo_storage/actions/workflows/ci.yml/badge.svg)](https://github.com/combo-lab/combo_storage/actions/workflows/ci.yml)
[![Hex.pm](https://img.shields.io/hexpm/v/combo_storage.svg)](https://hex.pm/packages/combo_storage)

Provides file storage integration for Combo.

## Installation

Add `:combo_storage` to the list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:combo_storage, "<requirement>"}
  ]
end
```
